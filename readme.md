# Babel lambda function
trained on microfilm data from 14th c. latin manuscripts

## arch
- sagemaker enpoint deployed in tensor env
- yolov5 model trained on 14th c. latin manuscritps
- classes include: text, image, margin

= sagemaker endpoint -> lambde function -> api endpoint -> node backend -> mongoDB

## params
example event body:
{
    "body": {
    "key": "msm854_fol. 47(v,r) expell_fol1L.jpg",
    "bucket": "msm854"
    }
}
